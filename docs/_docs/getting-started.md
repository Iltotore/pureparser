# Getting Started

## Dependency

Include PureParser in your project using your preferred build tool.

SBT: 

```scala 
libraryDependencies += "io.github.iltotore" %% "pureparser" % "version"
```

Mill:

```scala 
mvn"io.github.iltotore::pureparser:version"
```
